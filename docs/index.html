JumpForge 3DS

Build 2D platformer ideas directly on Nintendo 3DS hardware.

JumpForge 3DS is a homebrew game-making toolkit focused on fast level editing, sprite animation workflows, and immediate test-play loops.

  • Touch-first level and sprite editing
  • Animation tools and timeline controls
  • Platformer test mode (move, jump, dash)
  • Built with devkitPro/libctru
